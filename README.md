# node-mongo-boilerplate
Boilerplate code for Node.JS MongoDB fast Development

## Get started
```bash
git clone https://github.com/DivinIrakiza/node-mongo-boilerplate.git
```
```bash
cd node-mongo-boilerplate/
```
## Serve the app
```bash
yarn install 
//or
npm install
```
```bash
yarn start:dev
//or
npm run start:dev
```

Tada !
